# LOL 자동 게임 수락 매크로 (초보자 가이드)

이 프로그램은 리그 오브 레전드(LOL) 게임 매칭이 잡혔을 때, 화면에 나타나는 '수락' 버튼을 자동으로 클릭해주는 프로그램입니다.

## 사용해보고 싶다면? (for **windows**)

[다운로드 링크](https://github.com/PyeongBee/lol_auto_start_macro/raw/refs/heads/main/python/dist/AutoStart.exe)

## 다른 매크로 프로그램 개발로 응용하고 싶다면?

## 1. 파이썬(Python) 설치하기

이 프로그램을 개발하려면 컴퓨터에 파이썬이 설치되어 있어야 합니다.

1. [파이썬 공식 홈페이지](https://www.python.org/downloads/)에 접속합니다.
2. **Download Python 3.x.x** 버튼을 눌러 설치 파일을 다운로드합니다.
3. 설치 파일을 실행합니다.
   - **중요**: 설치 첫 화면 아래에 있는 **"Add Python to PATH"** 체크박스에 **반드시 체크**해주세요. (이걸 안 하면 명령어가 작동하지 않습니다!)
4. **Install Now**를 눌러 설치를 완료합니다.
5. (컴퓨터 재시작을 해야할 수도 있습니다.)

## 2. 소스 코드 다운로드

1. 이 저장소의 코드를 다운로드하거나 `git clone`을 합니다.
2. 다운로드한 폴더의 압축을 풉니다.
3. 폴더 안의 `python` 폴더로 들어갑니다. (`auto_start.py` 파일과 `start_button.png` 파일이 보여야 합니다.)

## 3. 필요한 라이브러리 설치하기 (pip install)

프로그램 실행에 필요한 도구들을 설치하는 단계입니다.

1. **명령 프롬프트(CMD)** 또는 **PowerShell**을 엽니다.
   - 윈도우 키 + R을 누르고 `cmd`를 입력 후 엔터를 치면 됩니다.
2. `auto_start.py` 파일이 있는 폴더로 이동해야 합니다.
   - 탐색기에서 `python` 폴더 주소창을 클릭해 주소를 복사합니다.
   - cmd 창에 `cd `를 입력하고 한 칸 띄운 뒤, 마우스 우클릭으로 주소를 붙여넣고 엔터를 칩니다.
   ```cmd
   cd C:\경로\...\python
   ```
3. 아래 명령어를 입력하여 필요한 라이브러리를 설치합니다.
   ```cmd
   pip install pyautogui opencv-python pillow
   ```
   - 설치가 완료될 때까지 잠시 기다립니다.

## 4. 개발하기

설치가 다 되었다면 코드를 수정하고 실행해볼 수 있습니다.

1. **코드 실행하기**
   터미널에서 다음 명령어를 입력하면 파이썬 스크립트가 바로 실행됩니다.

   ```cmd
   python auto_start.py
   ```

2. **코드 수정하기**
   - `auto_start.py` 파일을 메모장이나 코드 에디터(VS Code 등)로 엽니다.
   - 원하는 기능을 추가하거나 수정합니다.
   - 저장 후 다시 `python auto_start.py`를 실행하여 변경 사항을 확인합니다.

## 5. 실행 파일(.exe)로 바로 패키징하기

다른 사람에게 프로그램을 공유하거나, 파이썬 없이 실행하고 싶다면 `.exe` 파일로 만들 수 있습니다.

1. **PyInstaller 설치**

   아직 설치하지 않았다면, 아래 명령어로 패키징 도구를 설치합니다.

   ```cmd
   pip install pyinstaller
   ```

2. **실행 파일 만들기**

   - 폴더에 이미 설정 파일(`AutoStart.spec`)이 준비되어 있습니다.
   - 한 번 빌드를 하고나면 만들어지는 spec을 준비해두었습니다.
   - 바꾸고 싶을 땐 `AutoStart.spec`을 직접 바꾸시는 걸 추천드립니다.

   ```cmd
   pyinstaller AutoStart.spec
   ```

3. **결과물 확인**
   - 작업이 끝나면 `dist` 폴더가 생깁니다.
   - 그 안에 있는 `AutoStart.exe` 파일이 완성된 프로그램입니다!
   - `build` 폴더 안의 파일들로 만들어진 .exe 파일입니다.

## 6. 주의 사항 및 팁

- **이미지 인식**: 이 프로그램은 `start_button.png` 이미지를 화면에서 찾아서 클릭합니다.
  - 만약 해상도나 테마 문제로 인식을 못한다면, 본인의 화면에서 '수락' 버튼을 캡처하여 `start_button.png` 파일로 덮어씌워주세요. (이름은 똑같이 해야 합니다.)
- **관리자 권한**: 게임 클라이언트가 관리자 권한으로 실행된 경우, 이 프로그램도 **관리자 권한으로 실행**해야 클릭이 먹힐 수 있습니다. (cmd를 관리자 권한으로 열어서 실행하세요.)

## 7. 개발자를 위한 참고

- 사용 라이브러리: `pyautogui`, `tkinter` (내장), `threading` (내장)
- 이미지 인식을 위해 `opencv-python`이 필요합니다 (`confidence` 옵션 사용).
